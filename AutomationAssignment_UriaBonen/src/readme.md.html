1. Name - String+numbers

2.The user will move to “Thank you” page

Phone:050..
Email:test@..

The “contact us for help” frame will appear as long as the user doesn't arrive at the “contact us for potential customers” part (bottom  part on the page).

I wrote Automation Tests for most of the main scenarios according to my discretion and within the time I had.

package #2 (TestCases) contain all the Test Cases.
